# Contributing to SiliconFlow Image

## Quick Links

- **GitHub:** https://github.com/SonicBotMan/siliconflow-image
- **Issues:** https://github.com/SonicBotMan/siliconflow-image/issues

## How to Contribute

1. Bug fixes → Open a PR
2. New features → Open issue first
3. Questions → GitHub Issues

## Development

```bash
git clone https://github.com/SonicBotMan/siliconflow-image
cd siliconflow-image
pip install -r requirements.txt
python silflow.py --help
```
